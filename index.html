Inventory Management System

I developed an Inventory Management System for our Capstone Project using HTML, CSS, JavaScript, PHP, and MySQL. This system is designed to help manage and track inventory efficiently. It includes features such as adding, updating, and deleting inventory items, as well as generating reports and providing real-time inventory status. The front end is built with HTML, CSS, and JavaScript to ensure a user-friendly interface, while the back end uses PHP and MySQL for robust data management and processing.

Ticketing System

During my on-the-job training at GB Distributors, I was tasked with creating a Ticketing System with a chat feature. I used HTML, CSS, JavaScript, PHP, and MySQL to build this system. The Ticketing System was designed to manage customer support tickets efficiently, allowing users to submit, track, and resolve issues. The integrated chat system facilitated real-time communication between customers and support staff, improving the overall user experience. The combination of these technologies ensured a smooth and responsive interface, robust backend functionality, and effective data management.
